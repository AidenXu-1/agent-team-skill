# 安全说明

`agent-team` 是一个本地 Codex Skill。它的脚本不需要 API Key、账号密码或第三方服务凭证。

## 不要提交这些内容

- `.env` 或真实密钥。
- 私有项目里的 `docs/collaboration/` 产物。
- 用户聊天记录、客户资料、内部报告。
- 本地绝对路径中带有敏感信息的日志。
- 账号登录态、Cookie、Token。

## 如果发现安全问题

请优先通过 GitHub Security Advisory 反馈。  
如果无法使用 advisory，可以在 Issue 中只描述问题类型，不要贴出真实密钥、私有数据或可利用细节。

## 维护原则

- Scaffold 脚本默认不应访问网络。
- 验证脚本只应创建临时目录并检查生成物。
- 不应引入不必要的第三方依赖。
- 生成的协作文件可能包含项目隐私，不应被默认上传到公开仓库。
